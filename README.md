Detta projekt skapades genom att använda Angular CLI version 21.2.9.


**Webcourses** är en enklare webbsida uppbyggd med komponeneten courses, där det presenteras en tabell av kurser (courses.html). 

Länk: https://webcourses-ju2z.onrender.com


Sidans tabell listar kurser på webbutvecklingsprogrammet i Sundsvall, hämtade från ett APi med httpClient och get-anrop. 
Tabellen går att sortera i bokstavsordning efter kurskod, kursnamn samt progression via en drop-down meny.
Det går även att filtrera listan med hjälp av en sökruta där besökaren kan skriva in en sökfras.

Detta har löst med hjälp av signals samt computed-signals i Angular.
